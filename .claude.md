# Cat A Comb - Final Project Frontend

A React-based booking application for a cat grooming service, built with Vite.

## Tech Stack

- **Framework**: React 18.3.1
- **Build Tool**: Vite 6.4.1
- **State Management**: Redux Toolkit 2.6.1 & React Redux 9.2.0
- **Routing**: React Router DOM 7.4.0
- **Styling**: Styled Components 6.1.16
- **Additional Libraries**: React Scroll 1.9.3

## Key Files

### Configuration
- **vite.config.js**: Vite configuration with React plugin, port 5173, auto-open browser
- **eslint.config.js**: ESLint configuration for code quality
- **jsconfig.json**: JavaScript/TypeScript configuration
- **netlify.toml**: Netlify deployment configuration

### Entry Points
- **index.html**: Main HTML file, title: "Cat A Comb"
- **src/index.jsx**: React application entry point
- **src/App.jsx**: Root application component

### State Management
- **src/reducers/user.js**: User state reducer
- **src/reducers/treatments.js**: Treatments state reducer

### Utilities
- **src/utils/urls.js**: API endpoint URLs configuration

## Components Overview

### Pages
- **WelcomePage.jsx**: Landing page
- **Main.jsx**: Main application container
- **NotFound.jsx**: 404 error page

### Features
- **Booking.jsx**: Booking interface for grooming services
- **Cards.jsx**: Service cards display
- **Login.jsx**: User authentication
- **Logout.jsx**: Logout functionality
- **UserInfo.jsx**: User profile information
- **ValidationInfo.jsx**: Form validation feedback

### Layout
- **Header.jsx**: Application header
- **NavbarMenu.jsx**: Navigation menu
- **Footer.jsx**: Application footer
- **Loading.jsx**: Loading state component

### Hooks
- **useSticky.jsx**: Custom hook for sticky navigation behavior

### Styling
Components use styled-components with separate styling files:
- BookingStyling.js
- CardsStyling.js
- FooterStyling.js
- GlobalStyling.js
- HeaderStyling.js
- LoginStyling.js
- NavbarStyling.js
- UserInfoStyling.jsx
